Redis 教程

Redis 最典型的应用场景是作为分布式缓存。

学习 Redis，有必要深入理解缓存的原理，以及 Redis 作为一种缓存方案，在系统应用中的定位。

参考：缓存基本原理，有助于理解缓存的特性、原理，使用缓存常见的问题及解决方案。
